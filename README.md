# 🛒 Amazon Clone (HTML + CSS)

This is my **first front-end project**, where I recreated the Amazon homepage using only **HTML and CSS**.
The goal of this project was to practice core CSS concepts and build a real-world layout.

## 🧠 What I Learned

Through this project, I applied and strengthened my understanding of:

### 🔹 CSS Fundamentals

* CSS Syntax & Selectors
* Colors & Color Systems
* Text Styling
* Units (`px`, `em`, `rem`, `%`, `vh`, `vw`)

### 🔹 Box Model

* Margin, Padding, Border
* Height & Width
* Display & Visibility

### 🔹 Layout & Positioning

* Position (`static`, `relative`, `absolute`, `fixed`, `sticky`)
* Z-index
* Background images & properties

### 🔹 Flexbox

* Flex container & items
* `justify-content`, `align-items`, `flex-wrap`
* Responsive layout structuring

### 🔹 Advanced CSS

* Media Queries (basic responsiveness)
* Transitions & Transformations
* Animations

## ✨ Features

* ✅ Amazon-style navigation bar
* ✅ Search bar UI
* ✅ Hero section with banner
* ✅ Product grid layout (shopping section)
* ✅ Footer with multiple columns
* ✅ Hover effects
* ✅ Clean and structured layout using Flexbox
